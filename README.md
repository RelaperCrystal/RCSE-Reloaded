# RCSE 重置版

由 Avalon Edit 强力驱动，现在的 RCSE 已经支持语法高亮功能。另外，本代码被从头重写。

关于 RCSE - Remastered: 这个项目是在初期夭折的，最后选用了 WinForm 的原因是设计简单。本来想用 Ribbon 结果这个想法也夭折了。最后中规中矩的还是用了老的菜单栏设计。
